# Spam Detection System
SPAM email detection system using machine learning (Spacy)

# Setting up spacy
- In code repo:
  `pip3 install spacy`
-In directory create virtual environment by running:
  `python3 -m venv venv`
- and then activate the environment
  `source venv/bin/activate`
